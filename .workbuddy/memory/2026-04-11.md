# 2026-04-11 工作日志

## 修复：知识库页面两个前端 Bug

### 问题1 - 红框 "Cannot read properties of undefined (reading 'list')"
**根因**：`app.js` 的 `DOMContentLoaded` 回调中，`window.KnowledgeBaseAPI = KnowledgeBaseAPI` 赋值在 `App.init()` 之后。由于 `init()` 内部的 `navigate('knowledge-bases')` 是异步的（不 await），页面渲染时 `KnowledgeBaseAPI` 还没挂载到 `window`，导致 `window.KnowledgeBaseAPI.list()` 报错。
**修复**：将所有 `window.*API` 赋值移到 `App.init()` 之前。

### 问题2 - 左下角统计数据显示全为0
**根因**：`loadStatsData()` 使用 `|| 0` 处理默认值，如果后端返回数据中字段值为 `0`，逻辑上不会受影响（因为 `||` 对 0 确实会 fallback）。实际原因更可能是请求静默失败被 catch 吞掉。
**修复**：增加 API 就绪检查、改用 `?? 0`（nullish coalescing）避免语义混淆、增加详细 warning 日志便于调试。
